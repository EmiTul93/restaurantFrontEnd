VANTAGGI DELLE OTTIMIZZAZIONI:
------------------------------

1. MANUTENIBILITÀ
   - Codice CSS e JS centralizzato
   - Modifiche più facili e rapide
   - Meno duplicazione di codice

2. PERFORMANCE
   - File CSS/JS possono essere cachati dal browser
   - Caricamento più veloce delle pagine successive
   - Riduzione della dimensione dei file HTML

3. SCALABILITÀ
   - Facile aggiungere nuove pagine
   - Configurazione Tailwind riutilizzabile
   - Funzioni utility condivise

4. ORGANIZZAZIONE
   - Separazione chiara tra struttura (HTML), stile (CSS) e logica (JS)
   - File più leggibili e gestibili
   - Migliore esperienza di sviluppo
